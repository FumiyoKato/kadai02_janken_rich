## ①課題番号-プロダクト名
後出しじゃんけんアプリ

## ②課題内容（どんな作品か）
- じゃんけんができるアプリです
- 相手が後出しでじゃんけんの手を出しますが、その勝率は8割です
- 相手が勝つと「それは後出しじゃんけんだよ！」とコメントが出ます
- 自分が勝つかあいこになると、さまざまなビジネス用語がランダムに出てきます
